# GenSoftworks

Sieben KI-Agenten + ein Game Master erstellen ein GDD und eine WBB für ein Fantasy-Computerrollenspiel. Simulationsregeln: `simulation-2/game-master.md`. Kreative Vision: `simulation-2/briefing.md`.

## Pfade (relativ zu `simulation-2/`)

| Was | Pfad |
|-----|------|
| Briefing | `briefing.md` |
| Agent-Memory | `agents/{name}-memory.md` (finn, darius, emre, nami, vera, tobi, leo) |
| Agent-Roster | `roster/{vorname}-{nachname}.md` |
| Galerie | `gallery/concepts/`, `gallery/gdd/`, `gallery/wbb/` |
| Referenzbibliothek | `library/` (Repo-Root) |

## Dateinamen-Konvention

GDD: `gallery/gdd/KK-titel-vN.md` — WBB: `gallery/wbb/KK-titel-vN.md`

## Leitplanken

- **Deutscher Inhalt, englischer Code**
- **Echte Umlaute (ä ö ü ß)** — NIEMALS ae/oe/ue/ss
- **Niemals Quellen erfinden** — im Zweifel klar sagen
- **Briefing ist der Nordstern** — alle Artefakte orientieren sich an `briefing.md`
- **Genre ist Fantasy** — NICHT "dark fantasy"
- **Bilder: NIEMALS mit Text**
